Timer

The Timer class is an easy way to set up a countdown in your game.

new Timer(duration)

Source:
  • timing/Timer.js, line 10
Create a Timer.
Examples
import {Timer} from 'rythmoos-engine';
const myTimer = new Timer();
Parameters:
Name Type Description
duration number The duration of the timer, in milliseconds.

Members

(readonly) done :boolean

Source:
  • timing/Timer.js, line 53
Whether the timer is complete.
Type:
  • boolean
Example
// Create a 20 sec timer
const myTimer = new Timer(20000);

// In some game object's create method
myGameObject.create = () => {
  myTimer.start();
}

// Check whether it's done, in some game object's update method
myGameObject.update = () => {
  if (myTimer.done) {
    console.log('20 sec have elapsed');
  }
}

The time elapsed since the timer was started, equals to the duration once it is complete.
Type:
  • number
Example
const myTimer = new Timer(5000);

myTimer.start();

setTimeout(() => {
  console.log(myTimer.elapsed); // appr. 1000ms
}, 1000);
