Game

Game

The root and base class of any game.
You can override this class' methods or extend it to create your game.

Constructor

new Game(width, height, container)

Source:
  • containers/Game.js, line 42
Create a Game.
Examples

Overriding

const game = new Game(500, 500, document.body);

// Set scenes
game.scene = new SomeScene(game);

// Override method
game.create = () => {
  console.log('game being created');
}

game.update = () => {
  console.log('game updating!');
}

Extending

import {Game} from 'rythmoos-engine';

class AwesomeGame extends Game {
  create() {
    this.scene = new SomeScene(this);
    console.log('game being created');
  }

  update() {
    console.log('game updating!');
  }
}

new AwesomeGame(500, 500, document.body);

Methods

create()

Source:
  • containers/Game.js, line 186
Override this method.
Run when the game is create, after preload and load are complete.
You can for example set the game's starting scene from here.
Example
import {Game, Scene} from 'rythmoos-engine';

class AwesomeScene extends Scene {
  // My awesome scene
}

class AwesomeGame extends Game {
  // Add a scene when the game starts.
  create() {
    this.scene = new AwesomeScene(this);
  }
}

load()

Source:
  • containers/Game.js, line 164
Override this method.
Run after preload, and before the game starts.
You can load assets from here, files loaded in preload are now available.
Example
import {Game, Loader} from 'rythmoos-engine';

class AwesomeGame extends Game {
  // Load some assets before the game starts.
  load() {
    Loader.load('hero', 'hero.png');
    Loader.load('background video', 'background.mp4');
  }
}

preload()

Source:
  • containers/Game.js, line 145
Override this method.
Run first, before the game loads an starts.
Example
import {Game, Loader} from 'rythmoos-engine';

class AwesomeGame extends Game {
  // Preload JSON that will contain the links to our assets.
  preload() {
    Loader.load('sprite links', 'sprite-links.json');
    Loader.load('audio links', 'audio-links.json');
  }

  // Load the assets using the data we preloaded.
  load() {
    const sprites = Loader.get('sprite links');
    const audios = Loader.get('audio links');

    Loader.load('hero', sprites.hero);
    Loader.load('rock tile', sprites.rock);
    Loader.load('grass tile', sprites.grass);

    Loader.load('background music', audios.background);
    Loader.load('hero sound', audios.hero);
  }
}
