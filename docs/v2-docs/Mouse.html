Mouse

The mouse input.
It contains all information about the cursor and mouse buttons and wheel.

(static, readonly) cursorDetected :boolean

Source:
  • inputs/Mouse.js, line 225
Whether the cursor was detected.
This will be true once the cursor position is detected (first mousemove event).
Type:
  • boolean
Example
if (Mouse.cursorDetected) {
  // Cursor was detected
} else {
  // Cursor was not detected, Mouse.cursorX and Mouse.cursorY still have their default
  // values (0, 0).
}
