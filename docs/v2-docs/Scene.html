Scene

Methods

create()

Source:
  • containers/Scene.js, line 45
Override this method.
Run when the scene is created.
You can use this method to set your game objects.
Example

Adding game objects to the scene

import {Scene} from 'rythmoos-engine';
import {SomeGameObject, SomeOther} from 'wherever-they-are';

class AwesomeScene extends Scene {
  create() {
    this.gameObjects.someObj = new SomeGameObject(this);
    this.gameObjects.someOther = new SomeOther(this);
  }
}
