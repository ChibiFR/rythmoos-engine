GameObject

A game object is a game entity, usually containing graphics (e.g player character).

update()

Source:
  • containers/GameObject.js, line 168
Override this method.
Run every frame, before the rendering loop.
This occurs after the game and scene updates.
Examples

Rotating a game object

import {GameObject, Time} from 'rythmoos-engine';

class AwesomeGameObject extends GameObject {
  update() {
    // Rotate 20 degrees per second.
    this.rotation += 20 * Time.second;
  }
}

Moving a game object

import {GameObject, Time} from 'rythmoos-engine';

class AwesomeGameObject extends GameObject {
  update() {
    // Moving 30 pixels to the right per second..
    this.x += 30 * Time.second;
  }
}
