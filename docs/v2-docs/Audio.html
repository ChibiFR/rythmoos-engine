Audio

Audio

A playable audio file that can be analysed through its FFT data.
The instance inherits from the audio file passed in the constructor (loop, volume, etc).

Constructor

new Audio(audio, fftSizeopt)

Source:
  • media/Audio.js, line 13
Create an audio.
Examples
import {Audio} from 'rythmoos-engine';

Creating from an <audio> tag

const myAudioTag = document.querySelector('.my-audio-tag');
const myAudio = new Audio(myAudioTag);

Creating from a loaded audio

const myAudio = new Audio(Loader.get('my-audio'));
Parameters:
Name Type Attributes Default Description
audio HTMLAudioElement The audio file to use.
fftSize number <optional>

Examples

Getting the current time

// Assume our myAudio object has been playing for 15 seconds.

Setting the current time

myAudio.currentTime = 60; // Skips to 60 sec
console.log(myAudio.currentTime); // => 60

myAudio.currentTime -= 30; // Skips back to 30 sec
console.log(myAudio.currentTime); // => 30
