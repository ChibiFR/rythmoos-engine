Keyboard

Keyboard

The keyboard input.
It contains all information about the keyboard's keys state.
All keyboard events are listened, not regarding whether the game's canvas is focused.

(static) preventDefault :boolean

Source:
  • inputs/Keyboard.js, line 154
Whether to prevent the default key events' actions.
Type:
  • boolean
Examples
Keyboard.preventDefault = true; // Default key actions won't be fired
Keyboard.preventDefault = false; // Default key actions will be fired

Overriding native shortcuts

Keyboard.preventDefault = true; // Prevent the default actions

// Create our own "page refresh" shortcut using Ctrl+Alt+R instead of Ctrl+R
if (Keyboard.keyDown('ControlLeft', 'AltLeft', 'KeyR')) {
  // Reload the page if all 3 keys are down
  window.location.reload(true);
}

Whether the specified key(s) is/are down.

Checking for key down

import {Keyboard, Key} from 'rythmoos-engine';

}

Checking for shortcuts

import {Keyboard, Key} from 'rythmoos-engine';

if (Keyboard.keyDown(Key.ControlLeft, Key.KeyR)) {
  // Ctrl + R are down
}
