(static) requestBuffer(width, height) → {CanvasRenderingContext2D}

Source:
  • rendering/VirtualBuffer.js, line 51
Request the virtual context.
Make sure to set the right width and height required otherwise the context may fail at clearing itself properly.
If you don't know what width and height you are going to need, you can request arbitrary width and height and call ClearBuffer width the width and height that need to be cleared.

Name Type Description
